# 🏋️‍♂️ IronForge Fitness - Landing Page Profesional

¡Bienvenido a **IronForge Fitness**! Este proyecto es una landing page moderna, interactiva y totalmente responsiva diseñada para un gimnasio de alto rendimiento en Gran Canaria. 

Este trabajo ha sido desarrollado por **Izan Hernández Ojeda** como parte del módulo de Lenguajes de Marcas.

---

## 🚀 Características Principales

La web ha sido diseñada siguiendo los estándares más actuales de experiencia de usuario (UX) y diseño visual:

* **Diseño Full-Responsive:** Adaptado perfectamente a móviles, tablets y ordenadores usando el sistema de rejilla (Grid) de Bootstrap 5.3.
* **Navegación Fluida:** Menú persistente (*Sticky Navbar*) con efecto de *Scroll Suave* (Smooth Scrolling).
* **Componentes Avanzados:**
    * Carrusel de disciplinas configurado con tiempos de transición personalizados.
    * Sistema de rejilla complejo (6-3-1) para la sección de Servicios Premium.
    * Tabla de Horarios interactiva con ventanas emergentes (Modales) para cada clase.
    * Acordeón interactivo para la sección de Preguntas Frecuentes (FAQ).
* **Formulario de Contacto:** Maquetado profesionalmente utilizando *Floating Labels* y validación de campos.
* **Geolocalización:** Mapa interactivo de Google Maps integrado mediante `iframe`.
* **Efectos Especiales:**
    * Animaciones al hacer scroll mediante la librería **AOS (Animate On Scroll)**.
    * Iconos decorativos flotantes de fondo que imitan un efecto 3D.
    * Botón flotante de contacto directo por WhatsApp.

---

## 🛠️ Tecnologías Utilizadas

Para el desarrollo de este proyecto se han empleado las siguientes herramientas:

* **HTML5:** Estructura semántica de la página.
* **CSS3:** Estilos personalizados, variables de color y animaciones de levitación.
* **Bootstrap 5.3:** Framework principal para la maquetación y responsividad.
* **JavaScript:** Lógica para los modales, el carrusel y la inicialización de animaciones.
* **Google Fonts:** Tipografías *Oswald* (títulos) y *Roboto* (cuerpo).
* **FontAwesome:** Biblioteca de iconos vectoriales.
* **AOS Library:** Animaciones de entrada de elementos.
